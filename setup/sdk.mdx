## Overview
Use our lightweight SDK to initialize tracking and label key behaviors. During alpha, we provision access on a per-project basis.

<Steps>
<Step title="Request access">
  Reach out via the Support link in the navbar or email <a href="mailto:hi@mintlify.com">hi@mintlify.com</a> to join the SDK alpha.
</Step>

<Step title="We enable your project">
  We'll provision your project and send connection details and an API key.
</Step>

<Step title="Follow the integration guide">
  Once enabled, you'll receive the SDK install and initialization instructions tailored to your environment.

  <Card title="Use PostHog instead (stable)" icon="key" href="/setup/posthog">
    Prefer a production-ready path? Connect via PostHog today.
  </Card>
</Step>
</Steps>
