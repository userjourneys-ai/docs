<Note>
PostHog integration is required for the Visual Funnel Builder, available on the **Pro plan**.
</Note>

## Creating your API key


<Steps>
<Step title="Navigate to PostHog API settings">
  Go to <a href="https://eu.posthog.com/settings/user-api-keys" target="_blank" rel="noreferrer">eu.posthog.com/settings/user-api-keys</a>.

  <Tip>
  If your project is in the US, change the URL prefix to <code>us.posthog.com</code>.
  </Tip>
</Step>

<Step title="Click Create key">
  Open the <strong>Personal API keys</strong> section and select <strong>Create personal API key</strong>.

  <Frame caption="PostHog API Keys settings with Create personal API key button">
  <img src="/images/posthog_create_key_1.jpg" alt="PostHog API Keys settings page showing the Create personal API key button" />
  </Frame>
</Step>

<Step title="Fill in the form">
  - Select <strong>Projects</strong>
  - Choose the project you want to integrate

  <Frame caption="Create personal API key form with project selection">
  <img src="/images/posthog_project_selection_2.jpg" alt="Create personal API key form showing project selection and access tabs" />
  </Frame>
</Step>

<Step title="Set access permissions">
  Select **MCP Server** from the Scopes dropdown.

  <Frame caption="Select MCP Server from the Scopes dropdown">
  <img src="/images/mcp_server.png" alt="PostHog Scopes dropdown with MCP Server selected" />
  </Frame>

  Then add **WRITE** access for **Actions**.

  <Frame caption="Add WRITE access for Actions">
  <img src="/images/write_acces.png" alt="PostHog access settings showing write access for Actions" />
  </Frame>
</Step>

<Step title="Create and copy your API key">
  Click <strong>Create key</strong> and paste it into the form in your [project settings](https://app.userjourneys.ai/settings?tab=setup).
</Step>

<Step title="Copy Project ID from URL">
  Find your project ID in the URL. For example:

  ```text
  https://[eu/us].posthog.com/project/48510/settings/user-api-keys
  ```

  In this example, <code>48510</code> is the project ID. Copy your project ID into the form in your [project settings](https://app.userjourneys.ai/settings?tab=setup).
</Step>

<Step title="Test Connection">
  Click <strong>Test Connection</strong> to verify everything works. Replays will begin streaming immediately.
    <Frame>
  <img src="/images/posthog_test_connection_success.png" alt="A successful test connection message" />
</Frame>
</Step>
</Steps>

<Check>
You're all set
</Check>
